## 📖 Overview

**AI Gyan** is a dual-purpose platform:

1. **📰 Daily AI News Feed** — An autonomous agent scrapes, deduplicates, scores, and enriches the day's top AI news stories. Each story gets two summaries: one for developers, one for everyone else.
2. **🛠️ Resource Hub** — A curated, manually-maintained directory of the best AI tools, learning skills, and websites — all searchable from a single global search.

---

## ✨ Features

### News Feed
- 🤖 **Autonomous Agent** — Runs every 6 hours via Modal cron, no manual intervention
- 🔍 **Real-time Discovery** — Uses Gemini 2.0 Flash + Google Search Grounding
- 🧹 **Deduplication** — 7-day lookback window in Supabase to prevent repeated stories
- 🧑‍💻 **Dev Take / Casual Feed** — Toggle between technical and plain-English summaries
- 🏷️ **Auto-tagging** — LLM-driven tags (`#LLM`, `#NewModel`, `#GenerativeAI`, etc.)
- ⌘K **Global Search** — One shortcut to search across all news titles, sources, and tags

### Resource Hub
- 🛠️ **AI Tools** — ChatGPT, Claude, Cursor, Gemini, Hex, Napkin AI, Gamma, and more
- 📚 **Popular Skills** — Prompt Engineering, RAG, AI Agents, Fine-Tuning, MLOps, and more
- 🌐 **Powerful Websites** — Hugging Face, Papers With Code, arXiv, LMSYS Arena, and more
- 🔎 **Unified Search** — ⌘K searches both news and resources simultaneously
- ➕ **Easily Extendable** — Add new resources by editing `public/resources.json` (no code needed)

## ➕ Adding New Resources

All resources live in `public/resources.json`. Open it and add a new entry to any category:

```json
{
  "id": "unique-slug",
  "name": "Tool Name",
  "description": "A concise, accurate description of what this tool does.",
  "link": "https://example.com",
  "icon": "🔧",
  "tags": ["Tag1", "Tag2"],
  "featured": false
}
```

**Categories:**
| Key | Description |
|---|---|
| `ai_tools` | Apps, tools, and platforms powered by AI |
| `popular_skills` | Learning resources and skill guides |
| `powerful_websites` | Research hubs, leaderboards, and developer portals |

Save the file — the dev server hot-reloads instantly, no code changes needed.

## 🚀 Local Development

```bash
# Install dependencies
cd aigyan
npm install

# Start dev server
npm run dev
```

The site will be available at `http://localhost:5173`.

## 🤖 Agent Pipeline

The news agent runs autonomously on Modal. See [`AGENTS.md`](../AGENTS.md) at the repo root for the full architecture and setup guide.

```bash
# Test the agent locally
cd agent
source .venv/bin/activate
python test_agent.py

# Deploy to Modal (runs on 6-hour cron)
modal deploy agent.py
```
